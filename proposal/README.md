# Proposal Workspace

Draft proposal text and required supporting documents in the subfolders here.

Strategic resubmission planning notes that directly guide proposal rewrites should be stored in `proposal/narrative/`, including `proposal/narrative/fire_model_resubmission_strategy.md`.
Program-framing strategy memos that shape the rebid pitch, aims, and scientific narrative should also live in `proposal/narrative/`, including `proposal/narrative/fire_model_rebid_context_pitch_scientific_program.md`.
Cross-cutting narrative synthesis documents that consolidate arguments across theory, benchmarks, work plans, and deliverables should also live in `proposal/narrative/`, including `proposal/narrative/fire_model_proposal_integration_document.md`.

Numbered narrative snapshots and draft-comparison memos should be stored in `proposal/narrative/drafts/` so major conceptual revisions remain auditable over time.
