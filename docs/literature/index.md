# Literature

This section keeps the project's citation base visible and organized by role in the proposal.

## Why this section matters

The project draws from multiple literatures at once: wildfire spread physics, satellite fire datasets, fire regime statistics, fractal geometry, interface growth, complex systems theory, and landscape ecology. A dedicated literature section helps keep that intellectual map explicit.

The goal here is not to reduce the literature to a short reading list. Instead, this section keeps both the working bibliography and the intellectual structure of the reference base visible so the proposal can show how these bodies of work fit together.
