# Literature Map

This page groups the bibliography by the intellectual roles different papers play in the project.

## Core wildfire modeling references

These works define the operational and physics-based modeling context:

- Rothermel
- Finney / FARSITE
- Sullivan review
- WFDS, WRF-Fire, FIRETEC

## Data and event reconstruction

These references establish the observation and trajectory layer:

- MODIS burned area
- VIIRS active fire
- FireCCI
- MTBS
- Global Fire Atlas
- `FIRED`
- `CubeDynamics`

## Scaling and complex systems

These works provide the language for power laws, criticality, and scaling:

- Malamud
- Pueyo
- Telesca
- Clauset
- Sethna

## Geometry and rough interfaces

These references provide the fractal and interface-growth background:

- Mandelbrot
- Ricotta
- Plotnick
- Kardar-Parisi-Zhang
- Family and Vicsek
- Barabasi and Stanley
- Meakin

## Cross-scale ecological context

These works connect disturbance geometry to landscape structure and ecological process:

- Turner
- Peters and colleagues
- West, Brown, and Enquist
- Thompson

## Why this map matters

The project is strongest when these literatures are not treated as disconnected references. The proposal needs to show how they fit together:

data -> trajectories -> scaling diagnostics -> mechanism tests -> model benchmarks
