# Requirements Checklist

This page is the website view of the repository's working FIRE-MODEL compliance checklist.

!!! note
    This checklist is operational guidance for drafting. Final submission requirements must still be checked against the official solicitation and the current NSF PAPPG.

## Core proposal package

- Project Summary
- Project Description
- Intellectual Merit
- Broader Impacts
- Data Management Plan
- Postdoctoral Mentoring Plan, if applicable
- Budget and Budget Justification
- Biographical Sketches

## FIRE-MODEL framing checks

- The proposal clearly targets `FIRE-MODEL`
- The conceptual advance is explicit
- The computational framework is concrete
- The validation path is specific
- Cross-scale wildfire dynamics are addressed
- Relevant interacting processes are defined
- Interdisciplinary convergence is clear
- Practical value for prediction, management, or decision support is visible

## Source checks

- Verify the official FIRE solicitation page
- Verify the FIRE updates / FAQ page
- Verify the webinar materials
- Verify the current NSF PAPPG

## Submission timing to verify

- April 1 to April 7, 2026 window
- 5:00 PM local time deadline rule
