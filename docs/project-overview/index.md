# Project Overview

This section introduces the project as a proposal-centered research program rather than a collection of notes. It brings together the FIRE-MODEL call, the scientific motivation, and the organizational logic of the repository.

## What the project is about

At its core, this repository is developing an NSF wildfire-scaling concept: the idea that wildfire growth may follow reproducible geometric rules that emerge during fire expansion and connect local spread mechanisms to large-scale perimeter form.

The project sits at the intersection of:

- wildfire behavior and fire modeling
- scaling theory and fractal geometry
- remote sensing and event reconstruction
- spatiotemporal environmental data systems

## Why the repository is structured this way

The repository separates two kinds of work that proposals often blur together:

- `funder/` and funder-facing website pages explain what the call is asking for.
- `background_context/` and the theory pages explain the scientific argument we want to make in response.

That distinction matters because strong proposals need both:

1. compliance with the call, and
2. a clear intellectual position.
