# Wildfire Scaling: Rationale and Impact

This page explains why the wildfire scaling project matters, what it produces, who will use it, and why it is worth a multi-year investment.

## Why it matters here

Several pages in this repository describe theories, mechanisms, and tests. This page answers a different question: why this work is worth doing at all. It is useful when we need a human-readable explanation of the project's value for collaborators, reviewers, and future users.

## Core contributions

The project aims to deliver:

- a large-scale fire growth trajectory dataset
- open-source scaling diagnostics
- empirical characterization of wildfire perimeter scaling
- mechanistic modeling experiments
- model-evaluation benchmarks
- a theoretical framework for wildfire growth geometry

## Who benefits

- fire model developers
- remote sensing scientists
- fire behavior researchers
- Earth system scientists
- agency research groups and planning teams

Operational impact is expected to be indirect: better science can improve models, which can improve forecasting and decision tools over time.
