# Wildfire Scaling Work Plan

This page turns the wildfire scaling idea into an organized multi-year research program.

## Why it matters here

Several pages in this repository describe possible scaling laws, mechanisms, and open questions. This page shows how those ideas could become a concrete research program with sequencing, personnel roles, and deliverables. That makes it useful for proposal framing because it connects theory to execution.

## Core structure

The work plan is organized into four phases:

1. build the measurement system
2. identify the scaling regime
3. test mechanisms with simplified models
4. evaluate existing fire models and synthesize theory

## Why this helps the proposal

- it shows a credible path from satellite data to theory
- it demonstrates how `FIRED` and `CubeDynamics` support the workflow
- it creates a clear story for personnel roles and project sequencing
- it translates a geometric hypothesis into a defensible work plan
