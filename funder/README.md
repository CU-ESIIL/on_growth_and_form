# Funder Materials

Use this area for documents and guides that are directly about the funder and the call itself.

- `solicitation/`: official solicitation material plus funder-research guides such as the FIRE-MODEL briefing.
- `review_criteria/`: requirement summaries and compliance-oriented checklists.
- `templates/`: sponsor-provided templates and formatting examples.

This folder is intentionally separate from `background_context/`, which is reserved for scientific framing and conceptual synthesis.
