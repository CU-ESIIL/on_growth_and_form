# FIRE-MODEL Requirements Checklist

## Status

This checklist is a working aid derived from collected program information. It must be checked against the official solicitation and current NSF PAPPG before submission use.

## Core proposal package

- [ ] Project Summary
- [ ] Project Description
- [ ] Intellectual Merit framing is explicit
- [ ] Broader Impacts framing is explicit
- [ ] Data Management Plan
- [ ] Postdoctoral Mentoring Plan, if applicable
- [ ] Budget
- [ ] Budget Justification
- [ ] Biographical Sketches

## FIRE-MODEL framing checks

- [ ] Proposal clearly targets the `FIRE-MODEL` track
- [ ] Conceptual advance is explicit and not incremental
- [ ] Computational framework is clearly described
- [ ] Validation pathway is specific
- [ ] Multi-scale dynamics are addressed
- [ ] Relevant interacting processes are identified
- [ ] Convergence science and interdisciplinarity are explicit
- [ ] Connection to real-world wildfire prediction, management, or decision support is clear

## Process areas mentioned in the working briefing

- [ ] Fire physics
- [ ] Vegetation and fuel dynamics
- [ ] Atmospheric processes
- [ ] Climate interactions
- [ ] Infrastructure / built environment interactions
- [ ] Social systems or decision processes, where relevant

## Source documents to check

- [ ] Official FIRE solicitation / program page
- [ ] FIRE updates and FAQ page
- [ ] FIRE webinar materials
- [ ] Current NSF PAPPG

## Submission timing from the working briefing

- [ ] Confirm April 1 to April 7, 2026 submission window
- [ ] Confirm 5:00 PM local time deadline rule
