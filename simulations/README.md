# Simulations

## Self-contained model proof of concept

- `fire_poc/` contains a standalone local-first workflow for writing an `ELMFIRE` benchmark case, running a regime-aware comparison model, and producing a proposal figure without mixing exploratory machinery into the main proposal narrative directories.
- `fire_poc/WORKFLOW_BUILDOUT_2026-03-30.md` records the detailed build history, architecture, installed-toolchain notes, and current runtime settings for the proof of concept.
